Comment ça marche ?

Ce projet est une démonstration d’un assistant spécialisé sur Apple. Il utilise un backend Python hébergé sur un serveur (Raspberry Pi ou autre) qui se connecte à OpenRouter pour générer des réponses.

L’interface web est hébergée gratuitement sur GitHub Pages, tandis que le backend est sécurisé et exposé via un tunnel Cloudflare.

Ainsi, lorsque vous posez une question dans l’onglet “L’IA”, le site envoie votre texte au backend, qui le transmet ensuite au modèle d’IA.
